# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, assume good faith, and keep discussion about the work, not
the person. Harassment, insults, and bad-faith behaviour are not welcome and
will be moderated.

Report problems privately to the maintainer listed in [SECURITY.md](SECURITY.md).
Reports will be handled confidentially.
